SkyWatch API

A RESTful API for querying UFO sighting reports. Data sourced from the National UFO Reporting Center (NUFORC).

Version 1.0.0
https://api.skywatch.dev
🛸
Data provided by the National UFO Reporting Center (NUFORC). NUFORC has been collecting UFO sighting reports since 1974 and maintains the world's largest database of UFO encounters.

UFO Sightings

                
Retrieve and filter UFO sighting reports with pagination support.

                
                
                

                    

                        GET
                        /v1/sightings
                        List UFO sightings with filtering
                    

                    
                    

                        

                            
Query Parameters

                            
                            

                                
state string

                                
Filter by US state (2-letter code)

                            

                            
                            

                                
city string

                                
Filter by city name

                            

                            
                            

                                
shape string

                                
Filter by object shape

                            

                            
                            

                                
page integer

                                
Page number (default: 1)

                            

                            
                            

                                
per_page integer

                                
Records per page (1-100, default: 25)

                            

                        

                        
                        

                            
Response

                            
{
  "sightings": [
    {
      "id": 1,
      "date_time": "2023-12-15T20:30:00",
      "city": "Roswell",
      "state": "NM",
      "shape": "disk",
      "duration": "5 minutes",
      "summary": "Bright disk-shaped object hovering silently",
      "text": "I was driving home from work when I noticed...",
      "posted": "2023-12-16T10:15:00",
      "latitude": 33.3943,
      "longitude": -104.523
    }
  ],
  "total": 10,
  "page": 1,
  "per_page": 25,
  "pages": 1
}
